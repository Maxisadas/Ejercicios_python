Ejercicio 1

Se debe multiplicar dos números sin utilizar el signo de multiplicación, solo se deben aceptar valores enteros.

Código

def ejercicio1(a,b):
i = abs(b)
resultado = 0
positivo = abs(b) == b
c = a*-1 if positivo == False else a
while i>0:
    resultado = resultado + c
    i = i - 1
return 'El resultado de {}'.format(a) + ' por ' + str(b) + ' es ' + str(resultado)

Ejercicio 2

Código

def mayorNumero(array):
array.sort()
elem = array.pop()
return 'El numero mas grande de la lista es: '+str(elem)
